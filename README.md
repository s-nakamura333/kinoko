# 毒キノコ分類アプリ

毒キノコの物理的特性データから毒の有無を判定するためのデータセット分析とWebページです。  
教育機関や保育現場での毒キノコの安全確認に役立つよう設計しました。

## 🔧 使用技術
- Python
- Jupyter Notebook
